# Digital Marketing Portfolio - Zahid Abdillah

A modern, minimalist portfolio built with Astro 5 and Cloudflare. Featuring a custom Admin Dashboard for easy content management.

## 🚀 Features

- **Admin Dashboard**: Secure management of Profile, Projects, Experience, and Education.
- **Cloudflare Stack**: Powered by Cloudflare Pages, Workers, D1 Database, and R2 Storage.
- **Minimalist Design**: Optimized for readability and professional impact.
- **Print Friendly**: Perfect for physical CV/Resume distribution.

## 🛠️ Commands

| Command | Action |
| :--- | :--- |
| `npm run dev` | Start local development server |
| `npm run build` | Build for production |
| `npx wrangler d1 execute portfolio-db --local --file=migrations/0001_init.sql` | Run local migrations |
| `npx wrangler d1 execute portfolio-db --remote --file=migrations/0001_init.sql` | Run production migrations |
